# Phishing URL Detection System

## Overview
This project is a cybersecurity-focused web application that detects phishing URLs using heuristic analysis and pattern recognition techniques.

## Features
- URL structure analysis
- Suspicious keyword detection
- IP address-based URL detection
- HTTPS validation
- Pattern-based phishing indicators

## Tech Stack
- Python
- Flask
- HTML/CSS
- Rule-based detection logic

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000
